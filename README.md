# My App

A simple sign up app with Node.js backend.

## Setup

1. Install Node.js from https://nodejs.org/ or using winget: `winget install OpenJS.NodeJS`

2. Run `npm install` to install dependencies.

3. Run `npm start` to start the server on http://localhost:3000

## Features

- Sign up with name, email, password
- Stores users in users.json
- Serves the HTML page

Open index.html in browser or access via server.
